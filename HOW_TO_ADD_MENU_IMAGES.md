## üì∏ Quick Guide for Restaurant Owners

Adding beautiful food photos to your menu is now easier than ever! Follow this simple guide to upload images for your menu items.

## üöÄ Getting Started

### Step 1: Navigate to Menu Management

1. Log in to your restaurant owner account
2. Click on **"Owner Dashboard"**
3. Select your restaurant
4. Click on **"Menu Management"**

---

## üì§ Three Ways to Add Images

### Method 1: Upload from Your Computer (Recommended)

**Best for**: Photos you've taken or downloaded to your computer

1. Click **"Add Menu Item"** or edit an existing item
2. Scroll to the **"Menu Item Image"** section
3. Make sure the **"Upload File"** tab is selected
4. Click the **"Browse Files"** button
5. Select your image from your computer
6. Wait a few seconds for the upload to complete
7. You'll see a preview of your image
8. Fill in the other details and click **"Save"**

**‚ú® Pro Tip**: You can also drag and drop the image directly into the upload area!

---

### Method 2: Drag and Drop

**Best for**: Quick uploads when you have the file open

1. Open your file explorer/finder
2. Find the image you want to upload
3. Drag the image file over the upload area
4. The border will highlight - drop the file
5. Wait for the upload to complete
6. Preview appears automatically

**‚ú® Pro Tip**: This is the fastest way to upload multiple items!

---

### Method 3: Use an Image URL

**Best for**: Images already hosted online

1. Click **"Add Menu Item"** or edit an existing item
2. Scroll to the **"Menu Item Image"** section
3. Click the **"Image URL"** tab
4. Paste the image link (must start with http:// or https://)
5. Click **"Add"** button
6. Preview appears immediately

**Example URLs you can use**:
```
https://images.unsplash.com/photo-1546069901-ba9599a7e63c
https://images.unsplash.com/photo-1565299624946-b28f40a0ae38
```

## üéØ Image Requirements

### ‚úÖ What Works

- **File Types**: JPEG, PNG, WEBP, GIF, AVIF
- **File Size**: Up to 1MB (larger files are automatically compressed)
- **Filename**: English letters and numbers only
- **Dimensions**: Any size (we'll optimize it for you)

### ‚ùå What Doesn't Work

- PDF, Word documents, or other non-image files
- Filenames with special characters or non-English letters
- Broken or invalid URLs

---

## üîÑ Automatic Compression

### What Happens When Your Image is Too Large?

If you upload an image larger than 1MB, our system automatically:

1. **Compresses** it to a smaller size
2. **Converts** it to WEBP format (better quality, smaller size)
3. **Resizes** it to fit perfectly on the menu
4. **Notifies** you of the compression

**You'll see a message like**:
```
‚úÖ Success
Image uploaded and compressed from 2048KB to 856KB
```

**Don't worry!** The image will still look great - we use smart compression that maintains quality.

## üîß Managing Your Images

### How to Change an Image

1. Edit the menu item
2. The current image will show in the preview
3. Upload a new image or enter a new URL
4. The old image will be replaced
5. Click **"Save"**

### How to Remove an Image

1. Edit the menu item
2. Click the **[X]** button on the top-right of the image preview
3. The image will be removed
4. You can upload a new one or save without an image

## ‚ùì Common Questions

### Q: What if my image is too large?

**A**: No problem! Our system automatically compresses images larger than 1MB. You'll get a notification showing the original and final size.

### Q: Can I use images from Google?

**A**: You can use any image URL, but make sure you have the right to use the image. We recommend using your own photos or free stock photos from sites like Unsplash.

### Q: Why can't I upload my image?

**A**: Check these common issues:
- Is the file an image (JPEG, PNG, etc.)?
- Does the filename have only English letters and numbers?
- Are you logged in as a restaurant owner?

### Q: The image doesn't appear in the preview

**A**: Try these solutions:
- Check if the URL is correct and accessible
- Try uploading the file instead of using a URL
- Refresh the page and try again

### Q: How long does upload take?

**A**: Usually 2-5 seconds for most images. Larger images may take a bit longer due to compression.

## üÜò Need Help?

### If Something Goes Wrong

1. **Check the error message** - it usually tells you what's wrong
2. **Try a different image** - maybe the file is corrupted
3. **Refresh the page** - sometimes this fixes temporary issues
4. **Try a different browser** - Chrome or Firefox work best
5. **Check your internet connection** - uploads need stable internet

### Common Error Messages

| Error | What It Means | How to Fix |
|-------|---------------|------------|
| "Invalid file type" | File is not an image | Use JPEG, PNG, or WEBP |
| "Filename must contain only English letters" | Filename has special characters | Rename the file |
| "Invalid URL" | URL is not correct | Check the URL format |
| "Upload failed" | Network or permission issue | Check internet, try again |
